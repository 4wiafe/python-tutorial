# RPG Character Creator

## Overview

In this small Python project, I built a simple RPG character creator. The program validates the character’s name and starting stats, then generates a formatted character sheet showing strength, intelligence, and charisma using visual stat bars.

## What I Practiced

- Defining and using functions (`def`)
- Input validation with `isinstance()`
- Using conditional statements to enforce game rules
- Working with strings and formatting
- Creating visual stat bars using string multiplication
- Returning helpful error messages for invalid input
- Formatting output with f-strings
