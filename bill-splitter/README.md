# Python Bill Splitter (Basic Arithmetic Demo)

## Overview

In this small Python project, I practiced using basic arithmetic operations in Python. I calculated the total restaurant bill, added a tip, and split the final amount evenly among friends.

I also used the `round()` function to format the final amount each person pays.

## What I Practiced

- Variables for storing values
- Basic arithmetic operations (`+`, `*`, `/`)
- Updating variables using `+=`
- Calculating percentages (tip)
- Dividing a total bill among multiple people
- Using `round()` to format numbers
