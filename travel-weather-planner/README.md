# Python Travel Weather Planner (Conditional Logic Demo)

## Overview

In this small Python project, I practiced using conditional statements to make decisions in Python. The program checks different conditions such as distance, weather, and available transportation to determine whether a trip is possible.

## What I Practiced

- Boolean variables (`True` / `False`)
- Conditional statements (`if`, `elif`, `else`)
- Comparison operators (`==`, `<=`, `>`)
- Logical operators (`and`, `or`)
- Writing simple decision-making logic in Python
