# Python Caesar Cipher (Encryption & Decryption Demo)

## Overview

In this small Python project, I built a Caesar Cipher program that can encrypt text using a shift value. I also added input validation to ensure the shift value is valid and created separate helper functions for encryption and decryption.

## What I Practiced

- Defining and using functions (`def`)
- Creating reusable helper functions
- Input validation with `isinstance()`
- Using conditional statements
- String manipulation
- Using `str.maketrans()` and `translate()` for character substitution
- Implementing a basic encryption algorithm (Caesar Cipher)
