# Python Apply Discount Function (Functions & Validation Demo)

## Overview

In this small Python project, I practiced creating a function that calculates the final price of an item after applying a discount. I also added input validation to ensure the price and discount values are valid before performing the calculation.

## What I Practiced

- Defining and using functions (`def`)
- Returning values with `return`
- Input validation using conditional statements
- Checking data types with `type()`
- Using comparison and logical operators
- Performing percentage calculations
