# Python Report Card (Basic Data Types Demo)

## Overview

In this small Python project, I practiced working with basic Python data types. I stored simple student information and printed each value along with its data type.

I also used `isinstance()` to check whether a value belongs to a specific data type.

## What I Practiced

- Strings (`str`)
- Booleans (`bool`)
- Integers (`int`)
- Floats (`float`)
- Using `type()` to check data types
- Using `isinstance()` for type checking
